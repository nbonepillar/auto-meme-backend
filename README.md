# auto-meme-backend

A backend project built with NestJS and TypeScript.

## Installation

```bash
npm install
```

## Running the development server

```bash
npm run start:dev
```

## Build

```bash
npm run start
```

## Project Structure

- `src/` - Source code directory
- `dist/` - Compiled output
- `.env` - Environment variables (not committed)
